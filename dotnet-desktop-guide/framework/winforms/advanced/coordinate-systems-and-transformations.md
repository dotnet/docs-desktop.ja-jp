# <a name="coordinate-systems-and-transformations"></a>座標系と変換
GDI + は、描画する項目を変換 (回転、拡大縮小、翻訳など) できるように、ワールド変換とページ変換を提供します。 2つの変換を使用して、さまざまな座標系で作業することもできます。
