# <a name="using-a-brush-to-fill-shapes"></a>ブラシを使用した図形の塗りつぶし
GDI + <xref:System.Drawing.Brush> オブジェクトは、閉じた図形の内部を塗りつぶすために使用されます。 GDI + は、純色、ハッチパターン、画像テクスチャ、および色グラデーションのいくつかの塗りつぶしスタイルを定義します。
