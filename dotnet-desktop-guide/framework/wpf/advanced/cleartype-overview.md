 <span data-ttu-id="90786-109">[!INCLUDE[TLA#tla_winclient](../../../includes/tlasharptla-winclient-md.md)] で利用可能な ClearType は、Microsoft Windows グラフィックス デバイス インターフェイス (GDI) に含まれるバージョンにいくつかの改良が加えられた最新世代の ClearType です。</span><span class="sxs-lookup"><span data-stu-id="90786-109">The ClearType available in [!INCLUDE[TLA#tla_winclient](../../../includes/tlasharptla-winclient-md.md)] is the latest generation of ClearType which has several improvements over version found in Microsoft Windows Graphics Device Interface (GDI).</span></span>
